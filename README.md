# ibsol-wi2022-informationssicherheit-i

This is a recreation of my presentation for the course "Sicherheit in der Informationstechnik - Informationssicherheit I" in the [business informatics study program](https://www.ibs-ol.de/wirtschaftsinformatik) at [IBS IT & Business School Oldenburg](https://www.ibs-ol.de/).

To start the slide show:

- `pnpm install`
- `pnpm run dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes.

Learn more about Slidev at the [documentation](https://sli.dev/).
